# Responsive Layout Project

This project is a responsive webpage built using HTML5 and CSS3.  
It focuses on creating a layout that works well on mobile, tablet, and desktop screens using modern CSS techniques.

##  Color Theme
I chose purple color theme to create a clean and modern look and to learn more about how I can use this theme from my previous project to this one.  
CSS3 variables are used to manage colors, making it easy to update or change the theme across the entire site and exporing CSS variables as part of learning

##  Features
- Mobile-first design approach  
- Flexbox for navigation and content layout  
- CSS Grid for the gallery section  
- Media queries for different screen sizes  
- Smooth hover and transition effects  

##  Tools Used
- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  

##  How to Run the Project
1. Download or clone the project files.
2. Open the project folder.
3. Open the `index.html` file in any modern web browser.
4. Resize the browser window to see the responsive layout in action.

##  Purpose
This project was created as part of a class assignment to practice responsive web design using Flexbox, Grid, and media queries.
I am enjoy learning about flexbox and media queries
